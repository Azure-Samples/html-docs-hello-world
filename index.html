Azure App Service Web Apps

App Service Web Apps es una plataforma informática totalmente administrada y optimizada para alojar sitios web y aplicaciones web. Esta oferta de plataforma como servicio (PaaS) de Microsoft Azure le permite centrarse en su lógica empresarial mientras Azure se encarga de la infraestructura para ejecutar y escalar sus aplicaciones.
